Package Viewer アプリの使い方 (README)

概要

このウェブアプリは、あなたのAndroidデバイスにインストールされているアプリのパッケージ名リストを表示・管理するためのツールです。
データをアプリに登録するには、特定の形式でご自身のGmailアカウント宛にメールを送信する必要があります。

メール送信の要件

アプリが正しくデータを認識するためには、以下のルールに従ってメールを送信してください。

1. 宛先 (To)

あなた自身のGmailアドレス (xxxx@gmail.com)

2. 送信元 (From)

あなた自身のGmailアドレス (xxxx@gmail.com)

3. 件名 (Subject)

Android package list
この件名は完全に一致している必要があります。大文字・小文字も区別されます。

4. 本文 (Body)

メールの本文には、以下の形式のJSONデータのみを記述してください。余計な挨拶文や署名は含めないでください。

JSON形式:

[
  "デバイスのモデル名",
  "Android ID",
  [
    "パッケージ名1",
    "パッケージ名2",
    "パッケージ名3",
    ...
  ]
]

各要素の説明:

  • デバイスのモデル名 (文字列): データを識別するための任意の名前です。例: "Pixel 8", "Galaxy S24"
  • Android ID (文字列): デバイス固有のIDです。
  • パッケージ名リスト (文字列の配列): インストールされているアプリのパッケージ名のリストです。

具体的なメールの例

件名:

Android package list

本文:

[
  "Pixel 8 Pro",
  "1234567890abcdef",
  [
    "com.android.chrome",
    "com.google.android.gm",
    "com.google.android.apps.maps",
    "com.google.android.youtube"
  ]
]

推奨されるデータ送信方法

手動でメールを作成・送信することも可能ですが、必要な情報を自動で取得し、正しい形式でメールを送信するために、Automateアプリと連携することをお勧めします。
アプリのフッターに記載されているリンクから、対応するコミュニティフローをダウンロードしてご利用ください。
