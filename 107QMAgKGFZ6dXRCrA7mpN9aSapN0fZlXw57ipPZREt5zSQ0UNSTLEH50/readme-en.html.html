How to use Package Viewer App (README)

Overview

This web application is a tool to display and manage the list of package names of apps installed on your Android device.
To register data with the app, you need to send an email in a specific format to your own Gmail account.

Email Requirements

For the app to correctly recognize the data, please send the email according to the following rules.

1. To

Your own Gmail address (e.g., xxxx@gmail.com)

2. From

Your own Gmail address (e.g., xxxx@gmail.com)

3. Subject

Android package list
This subject must be an exact match. It is case-sensitive.

4. Body

The body of the email must contain only the JSON data in the following format. Do not include any extra greetings or signatures.

JSON Format:

[
  "Device Model Name",
  "Android ID",
  [
    "package.name.1",
    "package.name.2",
    "package.name.3",
    ...
  ]
]

Description of each element:

  • Device Model Name (String): An arbitrary name to identify the data. e.g., "Pixel 8", "Galaxy S24"
  • Android ID (String): The unique ID of the device.
  • Package Name List (Array of strings): A list of package names of the installed apps.

Specific Email Example

Subject:

Android package list

Body:

[
  "Pixel 8 Pro",
  "1234567890abcdef",
  [
    "com.android.chrome",
    "com.google.android.gm",
    "com.google.android.apps.maps",
    "com.google.android.youtube"
  ]
]

Recommended Data Submission Method

While it is possible to create and send emails manually, it is recommended to link with the Automate app to automatically retrieve the necessary information and send it in the correct format.
Please download and use the corresponding community flow from the link provided in the app's footer.
